# Post Formats

5 format templates. Pick format phu hop voi bucket + angle.

## Format 1: The Discovery (`tricks`, `innovation`, `tools`)

> **Hook:** "Tuan truoc minh thu [X] va..."
> **Body:** 
> - Van de: 1 cau
> - Cach thu: 2-3 cau (co the co code)
> - Ket qua: 1-2 cau voi number
> **CTA:** "Ai da thu chua? / Comment de minh share setup."
> **Media:** screenshot config hoac before/after metric.

**Example hook lines:**
- "Tuan truoc em thu Claude prompt caching — day la 3 thu em hoc"
- "Em vua setup sub-agent cho bot SME — day la pattern dang dung nhat"

## Format 2: The Story (`team`, `cases`)

> **Hook:** "Co 1 chuyen tuan nay lam team cuoi lan / bi team debate"
> **Body:**
> - Context: 1 cau
> - Thanh 1 ngo ngang hoac critical moment: 2-3 cau
> - Bai hoc: 1-2 cau
> **CTA:** "Minh se share full setup neu ai muon / tag 1 builder khac"
> **Media:** screenshot conversation hoac code snippet funny.

**Example hooks:**
- "Bot em ngao lan thu 3 trong tuan — lan nay no tu xoa skill chinh no"
- "Khach X vua demo bot phai sau 5 phut setup — team luong phan ung"

## Format 3: The Listicle (`tricks`, `insights`, `tools`)

> **Hook:** "3 thu em uoc biet som hon khi build [X]"
> **Body:**
> - Item 1: 1-2 cau + tip
> - Item 2: 1-2 cau + tip
> - Item 3: 1-2 cau + tip
> **CTA:** "Thu nao anh em da biet? Comment them."
> **Media:** numbered list graphic.

**Example hooks:**
- "5 mistake khi viet SKILL.md — em da mac het"
- "3 pattern prompt cache dang dung"

## Format 4: The Hot Take (`insights`)

> **Hook:** Bold statement, co the controversial.
> **Body:**
> - Thesis: 1 cau
> - Reasoning: 3-4 cau voi data
> - Counter: 1-2 cau acknowledge phia doi
> **CTA:** "Disagree? Comment."
> **Media:** quote card hoac chart.

**Example hooks:**
- "Flash model KHONG du cho enterprise proposal — day la ly do"
- "RAG dang bi overhype — context window giai quyet 80% case"

## Format 5: The Showcase (`community`, `team`, `cases`)

> **Hook:** Spotlight cu the vao 1 agent/skill/user.
> **Body:**
> - Gioi thieu: 1-2 cau
> - Highlight cu the (metric, feature, lessons): 2-3 cau
> - Link: 1 cau
> **CTA:** "Check thu — link [o day]."
> **Media:** screenshot agent output hoac demo GIF.

**Example hooks:**
- "Gioi thieu skill `finance-tracker` cua anh X — 50 install tuan dau"
- "Team RockShip vua ship `sme-social` — day la demo 30s"

## Pick Format by Bucket

| Bucket        | Recommended Formats           |
| ------------- | ----------------------------- |
| `tricks`      | Discovery, Listicle           |
| `innovation`  | Discovery, Hot Take           |
| `team`        | Story, Showcase               |
| `cases`       | Story, Showcase               |
| `community`   | Showcase                      |
| `insights`    | Hot Take, Listicle            |
| `tools`       | Discovery, Listicle           |

## Hashtag Rules

- Toi da 3 hashtag/post.
- Always include `#{brand_name}` hoac `#Rockship` (brand).
- 1 tech tag: `#ClaudeAI`, `#AIAgent`, `#Prompt`, `#MCP`, etc.
- 1 context tag neu relevant: `#BuildInPublic`, `#ViSME`, `#Startup`.
