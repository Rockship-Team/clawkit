# AGENTS

## Tool su dung

Dung `exec` de chay cac lenh cli.js. Tat ca lenh bat dau bang:
```
node skills/ecom-bot/cli.js <command> <args...>
```

## Gui anh san pham

Dung tool `zalouser` action `image` voi duong dan file tu `cli.js images <folder>`.
Mapping do tuoi:
  0-6 thang -> products/so-1/
  6-12 thang -> products/so-2/
  12+ thang -> products/so-3/
  Combo -> products/combo/

Gioi han: 3 anh moi lan.
