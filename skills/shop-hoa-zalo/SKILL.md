# Shop Hoa Zalo Bot

Bot tự động cho shop bán hoa qua Zalo cá nhân.

## Tính năng

- Tự động trả lời tin nhắn khách
- Báo giá theo catalogue sản phẩm
- Chốt đơn hàng
- Thông báo trạng thái giao hàng

## Cấu hình

Sau khi install, config sẽ nằm tại `config.json` trong thư mục skill.

Bạn sẽ cung cấp SKILL code thực tế cho phần này.
